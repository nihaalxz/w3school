W3.css

W3.CSS is a free, modern CSS framework that's responsive, easy to learn, and smaller and faster than other CSS frameworks. It's designed to simplify and speed up web development, and supports modern devices like mobile, laptop, tablet, and desktop. W3.CSS is standard CSS only, and doesn't require jQuery or JavaScript libraries.
