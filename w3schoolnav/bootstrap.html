BOOTSTRAP

Bootstrap is a free, open-source front-end framework for web developers that helps build responsive websites faster and easier. It was originally developed by a Twitter designer and developer in 2010 and was previously known as Twitter Blueprint. Bootstrap is built on HTML, CSS, and JavaScript and includes reusable code, design templates, and optional JavaScript plugins
